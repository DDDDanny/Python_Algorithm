# 链表

Linked List

#### 环境

Python 3.7

#### 内容

> 单向链表
>
> > + [x]  链表初始化
> > + [x] 从链表末尾添加节点
> > + [x] 从链表头添加节点
> > + [x] 在任意位置添加节点（根据data确定位置）
> > + [x] 在任意位置添加节点（根据索引index确定位置）
> > + [x] 删除任意节点
> > + [x] 更新节点

> 双向循环链表
>
> > + [x] 链表初始化
> > + [x] 从链表末尾添加节点
> > + [x] 从链表头添加节点
> > + [x] 在任意位置添加节点
> > + [x] 删除任意位置节点
